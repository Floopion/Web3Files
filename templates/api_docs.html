API Documentation

Routes

GET /countries - List all Countries
GET /countries/<optional:/count_id> - Get a Specific Country


POST /countries/<count_name&t; - Create a Specific Country


DELETE /countries/<count_id> - Delete a Specific Country

Optional parameters

count_id (string) Specific Country ID used to retrieve a single country
